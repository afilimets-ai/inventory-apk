# Add Room database schema with inventory entity models

## Overview

Define the core data layer using Room persistence library with entity classes for inventory items, categories, and locations. Include database migrations strategy and a repository pattern for data access.

## Rationale

An inventory management app is fundamentally data-driven. Establishing the data model and persistence layer early provides the foundation for all UI and business logic features.
