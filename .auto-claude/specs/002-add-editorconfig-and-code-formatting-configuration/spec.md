# Add EditorConfig and code formatting configuration

## Overview

Add an .editorconfig file and configure ktlint or detekt for consistent Kotlin code style across the project. Establishing formatting rules before any application code is written prevents style drift and reduces future reformatting churn.

## Rationale

Setting up code style enforcement early ensures all future contributions follow a consistent pattern from day one.
